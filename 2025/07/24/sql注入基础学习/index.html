sql注入基础学习

发表于 2025-07-24 更新于 2025-07-25

注入：通过构造一条精巧的语句，来查询得到想要的信息

注入分类： 1.按照查询字段： 字符型： "20"，“A” 数字型：23 2.按照注入方法： Union注入、报错注入、时间注入、布尔注入

3.注入点：可以实现注入的地方

判断注入方式

1.如何判断是数字型还是字符型：
注入id=1 and 1=2
有回显说明1=2成立，说明是字符型注入
无回显说明1=2不成立，说明是数字型注入

2.注入id=1 注入id=2-1
如果结果相同是数字型注入
结果不同字符型注入
（一般不用加号，会被理解为空格）

判断闭合方式

1.种类：
‘ “ ‘） “）

2.闭合的作用：
手工提交闭合符号，结束前一段查询语句，后面即可加入其他语句，，查询需要的参数，不需要的语句可以用注释符号’–+‘或’#‘或’%23‘注释掉
![[Pasted image 20250720144916.png]]

3.判断方法
可以直接再?id=1后面加\
分析报错信息：看\斜杠后面跟着的字符，是什么字符，它的闭合字符就是什么，若是没有，就为数字型。

4.判断查询列数
id=1’ order by 3 –+（字符）
id=1 order by 3 (数字)

总结：

1.查找注入点
2.判断是字符型还是数字型注入
3.如果是字符型，找到他的闭合方式
4.判断查询列数，group by(二分法)
5.查询回显位置
